## Descrição
> O projeto consiste no desenvolvimento de uma aplicação web destinada a auxiliar
produtores rurais no gerenciamento de rebanhos bovinos. A solução busca centralizar informações
sobre os animais, permitindo o cadastro e consulta de dados como identificação, raça, sexo, peso e
histórico sanitário.
> 
> A aplicação também possibilitará o controle de vacinas disponíveis e o registro das
aplicações realizadas em cada animal, garantindo um acompanhamento mais eficiente da saúde do
rebanho. Além disso, contará com funcionalidades de autenticação de usuários e controle de acesso,
promovendo maior segurança e organização das informações.
> 
> Com a digitalização desses registros, espera-se reduzir a dependência de anotações
manuais, minimizar erros no armazenamento de dados e facilitar a tomada de decisão dos
produtores rurais, contribuindo para uma gestão mais eficiente das propriedades agropecuárias.

## Motivação
> O projeto em questão teve seu desenvolvimento motivado pela convivência de um dos desenvolvedores com o público-alvo da aplicação. Este já havia estruturado o escopo do software e, devido à oportunidade repentina, optou por colocá-lo em prática.
